Letters

This script will plot the counts of letters either in the query string, or in the input box.

Grading stuff


Aesthetics

  • Changed axis text color
  • Changed axis text hover-over color
  • Thin grey y-axis grid lines
  • Sort alphabetically

C-Level

  • Pulls query string from URL, decodes it
  • Removes non-alphabet characters from query string, converts to lower case, and counts occurances
  • Outputs decoded text and couter to console
  • Displays non-interactive bar chart


B-Level

  • Page will by default load values from the query string. Populates chart as letters are typed into query box.
